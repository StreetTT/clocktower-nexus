# Apps

The `apps/` workspace contains the deployable application surfaces for Clocktower Nexus.

- `storyteller-console`: private Storyteller user interface
- `public-square`: shared public Town Square display
- `game-hub`: server-authoritative backend for state, sync, persistence, and APIs

These workspaces are intentionally placeholders in Task 1.1.2. Source code, tooling, and runtime wiring land in later tasks.
